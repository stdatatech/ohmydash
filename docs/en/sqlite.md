SQLite is a SQL database engine based database file.

#### Creation

1. Click '+' on the top right of data source list to open 'Datasource Wizard'.
2. Select 'SQLite', and then input a string as new SQLite data source name.
3. Click 'Click to upload file'ï¼Œselect sqlite database file from file system, click 'Next' to upload file. A generated sqlite datasource file name like 'sqlite-123456789.db' will be shown in last step which indicate SQLite Datasource is ready.
4. Click 'Next' to finish.
